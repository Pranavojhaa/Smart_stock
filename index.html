What The Project Does

SmartStock goes beyond forecasting alone. It turns retail demand predictions into inventory decisions that a planner or operations team can actually use.

Demand Forecasting

Predicts cumulative demand for the next 7 and 28 days using lag, rolling, price, calendar, and promo features across store-item combinations.

Inventory Planning

Converts forecasts into safety stock, reorder quantity, stockout risk, and overstock warnings using transparent business logic.

Portfolio Storytelling

Includes a Streamlit app, reusable Python package, README, tests, and business framing so the project looks polished in a portfolio review.

Why It Matters

This project was designed around a practical retail planning problem: how to avoid empty shelves without over-ordering slow-moving inventory.

Inventory Manager

  • See reorder priorities by item
  • Estimate stockout exposure
  • Plan replenishment with a safety buffer

Category Planner

  • Compare demand across categories
  • Inspect forecast horizons
  • Identify stronger or weaker product groups

Operations Analyst

  • Track forecast quality
  • Surface overstock risk
  • Review model lift against baselines

Tech Stack And Architecture

The repo is split into a reusable forecasting package, a data pipeline, tests, and a Streamlit app for interactive exploration.

src/smartstock/ data.py features.py modeling.py inventory.py pipeline.py app/ streamlit_app.py tests/ data, modeling, inventory

Python Stack

Pandas, scikit-learn, joblib, Streamlit, Plotly

Modeling Layer

Gradient boosting with horizon-specific targets and baseline comparisons

Deployment Note

This page is the Netlify-friendly project landing page. The interactive Python app itself is best deployed on Streamlit Community Cloud or another Python-capable host.

How To Run The Full App

Netlify now serves this landing page correctly. To run the interactive data app locally or on a Python host, use the Streamlit command below.

pip install -r requirements.txt PYTHONPATH=src python3 -m smartstock.pipeline PYTHONPATH=src streamlit run app/streamlit_app.py
